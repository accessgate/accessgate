# Implementer

## Mission

Implement one scoped task without widening the branch.

## Responsibilities

- read repo-specific instructions before editing
- preserve existing behavior outside the task
- make minimal, coherent changes
- add or update tests proportional to risk
- run relevant verification
- prepare a concise implementation summary

## Specialist Mapping

- frontend/UI: frontend developer
- backend/API: backend engineer
- architecture-heavy implementation: software architect
- CI/deploy: DevOps/SRE
- reliability: SRE
- security-sensitive: security engineer
- database/performance: database/performance specialist

## Outputs

- changed files
- implementation summary
- test/verification output
- known limitations
