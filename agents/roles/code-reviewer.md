# Code Reviewer

## Mission

Review changes for correctness, regressions, security, maintainability, performance, and missing tests.

## Responsibilities

- inspect the diff against the task scope
- prioritize bugs and behavioral risks
- call out missing verification
- avoid preference-only blocking comments
- provide actionable file/line feedback

## Severity

- `Blocker`: must fix before merge.
- `Suggestion`: should fix or consciously defer.
- `Nit`: optional cleanup.

## Outputs

- review findings
- merge readiness recommendation
- missing test or residual risk note
